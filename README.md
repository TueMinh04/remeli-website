# Remeli Website Demo

Bộ code website demo tĩnh cho Remeli.

## Cách chạy bằng VS Code

1. Giải nén file zip.
2. Mở thư mục `remeli-website` bằng VS Code.
3. Cài extension **Live Server** nếu chưa có.
4. Mở `index.html`.
5. Chuột phải và chọn **Open with Live Server**, hoặc bấm **Go Live** ở góc dưới VS Code.

## File chính

- `index.html`: cấu trúc website
- `styles.css`: giao diện, responsive layout, typography, motion
- `script.js`: scroll motion, popup modal, hiệu ứng túi đồ ăn, chuyển đổi VI/EN
- `assets/remeli-logo.png`: logo Remeli
- `assets/favicon.png`: favicon

## Ghi chú chỉnh sửa mới

- Title/slogan dùng font Anton.
- Body text giữ Be Vietnam Pro cho dễ đọc.
- Nút “Tham gia danh sách chờ” mở popup đăng ký.
- Nút VI/EN đổi nội dung website sang tiếng Việt hoặc tiếng Anh.
- Email liên hệ: Remeli.vn@gmail.com
